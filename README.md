 QuadraticSolver Suite — Решатель квадратных уравнений на 7 языках
QuadraticSolver Suite — это коллекция из семи независимых реализаций решателя квадратных уравнений вида ax² + bx + c = 0. Каждая версия работает на своём языке программирования и предлагает полный набор функций для нахождения корней (включая комплексные), с расширенными возможностями и удобным интерфейсом.

✨ Уникальные возможности (общие для всех версий)
📐 Решение квадратных уравнений с действительными и комплексными коэффициентами.

🧮 Вычисление дискриминанта и определение типа корней:

Два действительных корня (D > 0)

Один действительный корень (D = 0)

Два комплексно-сопряжённых корня (D < 0)

📥 Гибкий ввод коэффициентов:

Интерактивный режим (ввод с клавиатуры)

Поддержка аргументов командной строки (--a 1 --b -3 --c 2)

📤 Красивый вывод результатов с пояснениями.

🔁 Повторный расчёт без перезапуска (в большинстве версий).

💾 Сохранение истории вычислений (в некоторых версиях).

🌐 Поддержка дробных и отрицательных коэффициентов.

🚀 Быстрый старт

bash

C#

csc QuadraticSolver.cs && QuadraticSolver.exe
PHP

text

Дискриминант D = 1.0
